# Official Doc Sources

Use these primary sources first when confirming latest stable versions and setup instructions.

- Next.js: `nextjs.org/docs` and `github.com/vercel/next.js/releases`
- T3 Env (Next.js): `env.t3.gg/docs/nextjs`
- Clerk: `clerk.com/docs` and Clerk changelog/release notes
- Tailwind CSS: `tailwindcss.com/docs` and `github.com/tailwindlabs/tailwindcss/releases`
- shadcn/ui: `ui.shadcn.com/docs`
- Drizzle ORM: `orm.drizzle.team/docs` and `github.com/drizzle-team/drizzle-orm/releases`
- Neon: `neon.com/docs`
- PostgreSQL: `postgresql.org/docs`
- Vercel: `vercel.com/docs`
- TypeScript: `typescriptlang.org/docs` and `github.com/microsoft/TypeScript/releases`
- ESLint: `eslint.org/docs/latest`
- Prettier: `prettier.io/docs`
- AWS S3 / AWS SDK v3: `docs.aws.amazon.com` and `github.com/aws/aws-sdk-js-v3`
- Husky: `typicode.github.io/husky`
- lint-staged: `github.com/lint-staged/lint-staged`
- Upstash: `upstash.com/docs`

If a version is ambiguous across sources, prefer release notes and clearly state uncertainty before installing.
